HTML - Hyper Text Markup Language

Primeira Aula

  • "head: É usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site

  • "title: É usada como titulo do site, na ABA do navegador

  • "body: É o elemento no nosso documento responsável por ser o container de tudo que é conteúdo do site

  • "DOCTYPE: É a declaração deve ser a primeira coisa em seu documento HTML

  • "As Heading Tags (H1, H2, H3, ...) são recursos do HTML utilizados para destacar títulos e sub-títulos de uma página

  • "b: Apresentação típica seria negrito

  • "i: Apresentação típica seria italico

  • "blockquote: indica que o texto incluído é uma longa citação

  • "ul: representa uma lista de itens sem ordem rígida

  • "li: È usado para representar um item que faz parte de uma lista

  • "ol: È usado para representar representa uma lista de itens ordenados

  • "meta: Tags são linhas de código HTML ou "etiquetas" que, entre outras coisas, descrevem o conteúdo do seu site para os buscadores

  • "html: Abreviação para a expressão inglesa HyperText Markup Language, que significa Hipertexto de Marcação de Linguagem
